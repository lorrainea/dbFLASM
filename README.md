dbFLASM: Fixed length approximate string matching using a De Bruijn sequence
===

GNU GPLv3 License; Copyright (C) 2016 Lorraine A.K. Ayad, Solon P. Pissis.

Given a text T, pattern X, q-gram size q and error size k, dbFLASM computes the edit/Hamming distance of size k between q-grams of size q over alphabet Î£ in pattern X and text T using library libFLASM.
